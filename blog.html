Thoughts on Assignment 3

11/5/2018

What was challenging about the assignment, and what specific kinds of problems did I have. How did I solve those problems?

The most challenging for me was figuring out the click even registration. Historically, I've always used the onclick attribute for button tags, and so in this assignment since I couldn't modify the html I had to figure out how to register events through JS alone. Luckily this was covered in lecture, and everything else pretty much just fell into place.

What did I learn from this assignment? Were there any special insights you had? What did you find that you already knew?

Aside from the way to register events through JS, I learned a lot about manipulating nodes using JS. Specifically, how variables storing Nodes function as pointers to the specific node in the DOM, and if the node is moved the variable will stay pointing to that same node. You can break this with a clone function, but even still that new cloned element only exists as a single instance, and if you try and put it multiple places in the DOM it will be moved around, rather than duplicated.

What kinds of resources were helpful for completing the assignment? Specific websites? Lectures? The class Piazza forum? The TAs? How did I use each of these resources?

Again, the most helpful resources for me in figuring out this assignment was Google, StackOverflow, and the Mozilla Developer Network. In addition though, with this assignment, there were a couple concepts that were central to this assignment like the JS event registration that I learned in lecture, which was pretty nice because I've mostly already known the content covered in lecture. Learning a new concept through the lecture felt good.

What are one or two things you had to Google to complete the assignment?

I had to google how to remove elements from the DOM, as well as how to clone nodes for use elsewhere in the DOM. This came in useful for adding new sales posts to the site, as well as saving the original set of posts so I could accomplish the extra credit goals.
